# LangGraph Lab 

A collection of autonomous agents and agentic workflows built with LangGraph, focusing on state management, tool usage, and structured reasoning.

This repository serves as a modular playground for production-grade AI architectures, from real-time research agents to complex multi-step reasoning systems.

## Projects

| Project | Description | Stack |
|---|---|---|
| [agentic_search_graph](./agentic_search_graph) | ReAct agent with real-time web search, bilingual support, and source-traceable answers via Streamlit UI | LangGraph · Groq (Llama 3.3 70B) · Tavily · Streamlit |

## Structure

Each project lives in its own folder with independent dependencies, tests, and documentation.
